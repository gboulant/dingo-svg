# Drawing SVG sketchs

**contact**: [Guillaume Boulant](mailto:gboulant@gmail.com?subject=dingo-svg)

This package can be used to draw simple geometric 2D sketchs with a set of
turtle-like commands (move, line, circle, etc.) and save the resulting image
with the SVG format (vectorial drawing).

Explore the demo examples:

* [demos - d01.convexhull](demos/d01.convexhull): computation of a
  convex hull and visualisation in a SVG sketch;
* [demos - d02.surface3d](demos/d02.surface3d): isometric view of a
  function $z=(x,y)$;
* [demos - d03.remarkable](demos/d03.remarkable): drawing templates for
  the remarkable tablet;
* [demos - d04.guitarneck](demos/d04.guitarneck): drawing all the notes
  for all the strings and all the fret on a guitar neck;
